# 3D Skull Reconstruction from CT Imaging

## Overview
This project demonstrates the reconstruction of a 3D cranial model from CT scan data using 3D Slicer. The goal was to segment bone structures and generate a 3D visualization suitable for analysis and modeling.

## Tools Used
- 3D Slicer
- CT imaging data (.nrrd format)

## Process
1. Imported CT scan data into 3D Slicer
2. Created segmentation using thresholding to isolate bone tissue
3. Cleaned artifacts using manual editing tools
4. Generated a 3D surface model from the segmentation
5. Visualized the skull from multiple anatomical views

## Results
The final model successfully isolates cranial structures and provides a clear 3D representation of the skull. Minor surface layering artifacts are present due to CT slice resolution, which is typical in medical imaging datasets.
